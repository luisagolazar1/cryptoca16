# CRYPTOCA16 🚀

**Advanced Cryptocurrency Analysis System**

Una plataforma moderna y profesional para el análisis en tiempo real de criptomonedas, inspirada en el algoritmo FXCA16.

---

## 🌟 Features

- ✅ **Dashboard en Tiempo Real** - Datos actualizados cada 6 horas desde Binance
- ✅ **Gráficos Interactivos** - Visualización de precios y volatilidad con Recharts
- ✅ **Análisis de Mercado** - Estadísticas de volatilidad, sentimiento bull/bear
- ✅ **Filtros Dinámicos** - Todas, Ganadores, Perdedores (24h)
- ✅ **Diseño Responsivo** - Optimizado para móvil, tablet y desktop
- ✅ **UI/UX Profesional** - Dark mode con gradientes y animaciones

---

## 🛠️ Tech Stack

- **Framework**: Next.js 14
- **Styling**: Tailwind CSS 3.3
- **Charts**: Recharts 2.10
- **Deployment**: Vercel
- **Data Source**: Binance API

## 🔄 Actualización de Datos

Los datos se actualizan automáticamente cada 6 horas mediante GitHub Actions:

```
.github/workflows/update-data.yml
├── Fetch data from Binance API
├── Analyze cryptos
└── Auto-commit & push to main
```
